「list_variable.sort()」是一個用在 list_variable 的 function，只能排序 item 皆為同 type 的 list。

對 string list 直接使用「.sort()」的話，大寫字母排在小寫字母前，同大小寫的字母之間再比較字母本身在a~z的位置。

「.sort(key=str.lower)」可以忽略大小寫差異，直接照在a~z的順序排。

「.sort(key=str.lower, reverse=True)」其中「reverse=True」代表將現在的 list 反向重排一次。

還有另一種寫法是「sorted(list_variable, key=str.lower, reverse=True)」

可以對數字型態的 list 使用「sum(list_variable)」加總。
