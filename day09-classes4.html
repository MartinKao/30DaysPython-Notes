可以將設定好的 class 存在另一個 xxx.py 檔(稱為module)，然後要用的話就「from xxx import some_class」。

執行檔只要 from xxx ，不用加 .py。import class 的名稱就好，後面不用加()。

放在同一層一料夾裡。


另一種方法，先創一個資料夾，資料夾裡面要有一個叫做「__init__.py」的檔案，和其他放 module.py 的檔案。

「__init__.py」裡面什麼都不用寫。只要有這個檔案存在，代表他的上層資料夾可以被視為一個 module。

執行檔用「from folder.module import class, function」。
