上一節的 email 是純文字訊息，要傳 html 訊息的話必須：

 

「from email.mime.multipart import MIMEMultipart」。

 

「from email.mime.text import MIMEText」。


「the_msg = MIMEMultipart("alternative")」代表這段文字是 HTML 格式。

「MIMEText(some_txt,'plain')」代表將 some_txt 轉換成純文字。

「MIMEText(some_txt,"html")」代表將 some_txt 轉換成 html 格式。

string 物件加上「.attach(other_string)」可以附加另一段上去。
