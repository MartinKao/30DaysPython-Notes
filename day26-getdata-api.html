要使用 key 認證，必須「import requests」「from requests_oauthlib import OAuth1」

「auth = OAuth1('app_key','app_secret','user_oauth_token','user_oauth_token_secret')」。

用 dictionary 值建立要搜尋的 params，可以使用的 key 的名稱會在網站上有說明。

之後建立 request 「r = requests.get(url,auth=auth, params=params)」

設計成 function 之後，可以「return r.json()」。之後就可以透過 dictionary key 來找到想要的資料。

若是有不確定的 key 名稱，可以使用「xxx.get('key_name')」，找不到則 return None。一般的「xxx['key_name']」則會發生 error。
